Query Modifiers

limit() restricts number of documents; sort() orders results.

// Return only 5 documents
db.users.find().limit(5)

// Sort by age descending
db.users.find().sort({ age: -1 })

Indexing Basics

Indexes improve query performance. Create them on fields used frequently in queries.

// Single-field index
db.users.createIndex({ email: 1 })

// Compound index
db.orders.createIndex({ customerId: 1, createdAt: -1 })

Backup & Restore

// Backup (mongodump)
mongodump --uri='mongodb://localhost:27017' --out=./dump

// Restore (mongorestore)
mongorestore --uri='mongodb://localhost:27017' ./dump
